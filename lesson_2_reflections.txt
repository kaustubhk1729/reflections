What happens when you initialize a repository? Why do you need to do it?

Transform the current directory into a Git repository. This adds a .git subdirectory to the current directory and makes it possible to start recording revisions of the project.It can be used to convert an existing, unversioned project to a Git repository or initialize a new empty repository.The .git subdirectory contains all of the necessary metadata.
